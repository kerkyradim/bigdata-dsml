# Δήλωση χρήσης LLM — Εξαμηνιαία Εργασία DSML 2026

**Ονοματεπώνυμο:** Κερκύρα Δημησιάνου 

| Ενότητα εργασίας | Εργαλείο | Έκταση χρήσης | Τι έγινε χωρίς LLM |
|---|---|---|---|

| **Setup περιβάλλοντος (WSL, Spark, VPN, kubectl)** | Ίδιο εργαλείο | Καθοδήγηση βημάτων setup βάσει των οδηγών του μαθήματος | Η πραγματική εγκατάσταση, η σύνδεση VPN, οι εκτελέσεις spark-submit και ο έλεγχος pods/logs |
| **Query 1 — κώδικας** (`Q1_df.py`, `Q1_df_udf.py`, `Q1_rdd.py`, `Q1_convert_parquet.py`, `Q1_df_parquet.py`) | Claude, Gemini | Βοήθησε στη δημιουργία του βασικού σκελετού των scripts (δόμηση αρχείου, `argparse`, `perf_counter`, paths HDFS και στην βασική ροή read → filter → aggregate → write) και στην διαχείρηση των δεδομένων με parquet | Η λογική του Query 1 (στήλες `Premis Desc` / `TIME OCC`, κανόνες τμημάτων ημέρας, υπολογισμός ποσοστών), οι διορθώσεις, οι εκτελέσεις στο cluster, η επαλήθευση αποτελεσμάτων και η ανάλυση χρόνων |
| **Query 2 — κώδικας** (`Q2_df.py`, `Q2_sql.py`) | Ίδιο εργαλείο | Βοήθησε στον αρχικό σκελετό (SparkSession, ανάγνωση CSV,  timing, αποθήκευση αποτελεσμάτων) | Ο σχεδιασμός του Query 2 (parsing `DATE OCC`, `row_number` ανά έτος,  SQL βήματα), οι δοκιμές, τα αποτελέσματα και η σύγκριση επιδόσεων |
| **Αναφορά — Query 1 & Query 2** | Ίδιο εργαλείο | Βοήθησε στη διαμόρφωση κειμένου και στη δομή ενοτήτων (πίνακες, συμπεράσματα) με βάση τα δικά μου μετρημένα δεδομένα | Η τελική επεξεργασία, η επιβεβαίωση αριθμών από τα log files, και η προσωπική ερμηνεία των αποτελεσμάτων |
| **Query 3 — κώδικας** (`Q3_df.py`, `Q3_rdd.py`, `Q3_df_joins.py`) | Ίδιο εργαλείο | Βοήθησε στον αρχικό σκελετό (SparkSession, paths HDFS, `argparse`, `perf_counter`, βασική ροή read → aggregate → join → write), στην ιδέα χρήσης `reduceByKey` αντί `groupBy` για αποφυγή OOM στο GeoJSON, στην δόμηση κα στην ερμηνεία των joins | Η λογική του Query 3 (τύπος per-capita, parsing income CSV), οι εκτελέσεις benchmark (3 executors) |
| **Αναφορά — Query 3**  | Ίδιο εργαλείο | Βοήθησε στη διαμόρφωση κειμένου και στους πίνακες με βάση τα δικά μου μετρημένα δεδομένα | Η τελική επεξεργασία, η επιβεβαίωση αριθμών από logs/`Q3_times.txt`/`Q3_join_times.txt`, και η προσωπική ερμηνεία |
